# ForensiX — Medico-Forensic DBMS

## 📌 Project Overview
**ForensiX** is an enterprise-grade, role-based Medico-Forensic Database Management System engineered for hospital Forensic Medicine Departments (FMD). Built with Django 5.2 and MySQL 8.0, the system streamlines the recording and tracking of sensitive medico-legal cases, postmortem examinations, clinical findings, evidence chain of custody, and court reports.

---

## 🚀 Key Features

* **🛡️ Role-Based Access Control (RBAC):** Tailored dashboards and granular permission controls for 4 primary user roles:
  * **Administrator:** System audit logs, user management, and department metrics.
  * **Doctor / JMO:** Case creation, clinical examinations, postmortems, and court report generation.
  * **Lab Technician:** Evidence tracking, laboratory test processing, and chain-of-custody logging.
  * **Viewer:** Strict read-only access with sensitive Patient PII protection.
* **🗄️ Normalized MySQL 3NF Architecture:** 17 normalized tables with constraints, stored procedures, 12 custom views, and 8 automated triggers.
* **🔒 Evidence Chain of Custody:** Tracks barcode/QR IDs, handling history, and storage parameters for forensic specimens.
* **✨ Modern Glassmorphism UI:** Custom responsive web interface featuring dynamic KPI cards and real-time toast notifications.
